# Wireguard-WUI

Wireguard-WUI est une interface web (Web User Interface) permettant de gérer son VPN Wireguard, la configuration du serveur, les clients, ect.. de manière simplifiée et plus agréable que vie le terminal.

Stack technique :

- Frontend : html, css, JS, Handlebar
- Backend : NodeJS Express
- BDD : SQLite
